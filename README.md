## What is Quantum?

Quantum is a **native AI-powered IDE for macOS** built from the ground up with **SwiftUI** and **Apple's own frameworks** ‚Äî no Electron, no web views, no compromises.

While other editors pretend to be Mac apps, Quantum breathes the same air as Finder, Xcode, and the Dock. It respects your RAM. It respects your battery. It respects *you*.
